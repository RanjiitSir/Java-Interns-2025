
							Programs on BASIC I/O

Create a program to add two numbers provided by the user and display the result.

Write a program to accept marks of 3 subjects and calculate and display total and percentage.

Write a program which accept radius of circle from user and calculate its area. Consider value of PI as 3.14.

Write a program which accept width & height of rectangle from user and calculate its area. (Area = Width * Height)

Write a program which accept distance in kilometer and convert it into meter.
(1 kilometer = 1000 Meter)

Write a program which accept temperature in Fahrenheit and convert it into Celsius.
(1 Celsius = (Fahrenheit - 32) * (5.0/9.0))

Write a program that accepts two numbers from the user and swaps their values.

Write a program that accepts two numbers from the user and swaps their values.
(Without using a third variable)
